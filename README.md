# 🐍 Bootcamp Python para SIG

Repositorio del Bootcamp de Python orientado a Sistemas de Información Geográfica (SIG).

## Objetivo

Aprender Python de forma práctica para automatizar procesos geoespaciales, analizar datos espaciales y desarrollar aplicaciones SIG.

## Tecnologías

- Python
- Git
- GitHub
- Visual Studio Code
- Pandas
- GeoPandas
- Shapely
- Rasterio
- PostgreSQL / PostGIS
- FastAPI
